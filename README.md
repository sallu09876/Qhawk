# 🚀 Qhawk

Enterprise B2B Management System

---

## 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/Qhawk.git
cd Qhawk

# Install root dependencies
npm install

# Install Client dependencies
cd Client
npm install
cd ..

# Install Server dependencies
cd Server
npm install
cd ..
```

---

## ▶️ Run the Project

Start the development server from the root directory. This will concurrently run both the Client and Server:

```bash
npm run dev
```

Then open your browser and go to:

```
http://localhost:5173
```

---

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS 
* Node.js / Express (Backend)
* PostgreSQL (Database)
